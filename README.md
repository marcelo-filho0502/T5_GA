# T5 - Coloração de Grafos com DSatur

Este projeto é o **Trabalho Prático 5** da disciplina de **Resolução de Problemas com Grafos**. O objetivo é modelar o mapa político do Brasil como um grafo não direcionado e aplicar a heurística **DSatur** (Degree Saturation) para obter uma coloração válida dos estados utilizando um número reduzido de cores.
